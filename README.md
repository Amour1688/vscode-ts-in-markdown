## Note

* Enable completion

Markdown by default disables quick suggests on normal words. It can be re-enabled using

```json
{
  "[markdown]": {
    "editor.quickSuggestions": {
      "other": true,
      "comments": true,
      "strings": true
    }
  }
}
```
